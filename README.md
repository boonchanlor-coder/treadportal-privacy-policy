# Privacy Policy for TreadPortal

TreadPortal respects your privacy.

## Information We Collect
TreadPortal may collect basic user information such as:
- Account information (username, email)
- Profile content provided by the user
- Photos or media uploaded by the user

## Camera Permission
TreadPortal may request access to the device camera for features such as:
- Uploading profile photos
- Posting images

The camera is only accessed when the user chooses to use these features.

## How We Use Information
Information is used only to provide core app functionality and improve the user experience.

## Data Sharing
We do not sell user data to third parties.

## Data Security
We take reasonable measures to protect user information.
